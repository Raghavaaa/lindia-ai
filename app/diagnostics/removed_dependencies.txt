PACKAGES TO POTENTIALLY KEEP
-----------------------------
May be needed for document processing if backend handles uploads:
- pillow (if image uploads)
- PyPDF2 (if PDF parsing in backend)
- pdf2image (if PDF preview generation)

Decision: Remove all unless explicitly needed for backend-specific document handling.

ESTIMATED SIZE REDUCTION
------------------------
Before: ~3.5GB+ (with all ML packages)
After: ~200MB (minimal backend dependencies)
Reduction: ~95% smaller Docker image

REPLACEMENT STRATEGY
--------------------
All AI/ML operations are delegated to AI_ENGINE_URL via HTTP API calls.
Backend focuses on:
- API routing and request handling
- Database operations
- Authentication/authorization
- Business logic
- API response formatting
